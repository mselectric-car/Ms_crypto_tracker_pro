# MS Crypto Tracker Pro v3.1

Stabilna verzija prilagođena uploadu sa Android telefona.

## Funkcije
- pretraga velikog broja kriptovaluta
- watchlista
- portfolio i P/L
- grafikoni 1D, 7D, 30D, 90D i 1Y
- TradingView
- RSI i momentum tehnička procjena
- Fear & Greed Index
- alarmi pri otvaranju i osvježavanju
- tamna, svijetla i sistemska tema
- četiri boje aplikacije
- EUR, USD i GBP
- APK i AAB build

Tehnička procjena nije finansijski savjet.
